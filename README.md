# Dart Learning

Repo d'apprentissage Dart - mini-projets réalisés en autodidacte.

## Projets

| Projet | Description | Concepts |
|--------|-------------|----------|
| [exercices-syntaxe](./exercices-syntaxe) | Syntaxe de base | Variables, types, classes, enums, mixins, records |
| [cli-todo](./cli-todo) | App CLI todo list | dart:io, stdin, switch, List |
| [bibliotheque](./bibliotheque) | Gestion de bibliothèque | POO avancée, héritage, tests unitaires |

## Setup

```bash
cd <projet>
dart pub get
dart run
```

## Objectif

Apprendre Dart avant de passer à Flutter.
