# CLI TODO

Application CLI simple de gestion de tâches.

## Concepts appris

- Import `dart:io` pour les entrées/sorties
- `stdin.readLineSync()` pour lire l'input utilisateur
- `switch` expression (Dart 3)
- Manipulation de `List`
- Boucle `while`

## Utilisation

```bash
dart run cli_todo.dart
```
